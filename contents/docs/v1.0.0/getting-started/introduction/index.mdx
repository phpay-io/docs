With PHPay you can implement payments with different gateways simply and easily with an elegant and straightforward unified interface.

## Getting Started

To begin using the PHPay, follow these simple steps:

- install library into projet with:

```bash
composer require phpay-io/phpay
```

## Conclusion

PHPay simplifies the integration with multiple payment gateways, offering a clean and unified interface to handle various payment operations effortlessly. With just a few steps, developers can set up the library, manage client information, and generate transactions like invoices and QR codes. By leveraging PHPay, you streamline your payment process, saving time and ensuring a more elegant implementation in your projects.
